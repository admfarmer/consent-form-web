# ConsentFormWeb

## Installation

```
git clone https://github.com/admfarmer/consent-form-web.git consent-form-web
cd consent-form-web
npm i

nano src/environments/environments.prod.ts
nano server.js

```

## Running

```
npm run build
```

## PM2

```
pm2 start server.ks --name consent-form-web
```
